Quais as principais tipos de tags html?

Tags Inline

Tags Inline são tags que não modificam a estrutura html, você pode utiliza-la dentro de outras tags e ela não vai "quebrar" a estrutura do texto.

Algumas tags inline:

spam

Ela é uma tag do tipo container generico inline, ela pode ser usada para agrupar elementos para fins de estilos usando os atributos class ou id. Ela deve ser usada apenas quando nenhum outro elemento semantico for apropriado.

code

A tag code é uma tag do tipo inline, ela é usada para especificar código no texto, como por exemplo: nome = 'iarlen'.

b

É uma tag inline também utilizada para mudar o estado fisico de um texto, ao usar a tag b o texto ficará em negrito.

i

A tag i é usada para deixar o texto em italico e como as outras Tags citadas acima, ela também é inline.

Tags block

Tags em block ou blocos, são tags que não podem ser usadas dentro de outras tags, pois ela destrói a estrutura do texto. Ela cria uma especie de bloco ou container que podem armazenar Tags do tipo inline.

Exemplos de Tags Block

div

A Tag div é um container em forma de bloco, semelhante ao spam, porém, não pode ser usada dentro de outras Tags, pois ela "destrói" a estrutura do texto. Ela pode ter atributos do tipo class ou id, assim como o spam.

blockquote

É um tipo de bloco para se usar em textos de citações, assim como as demais tags block, não pode ser usado dentro de outras tags, ele é semelhante a tag inline q.
